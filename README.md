# market-sentinel
A full-stack financial infrastructure tool for high-impact news surveillance.

## Architecture
- **Engine:** Python script using `httpx` for stealth API interception.
- **Broker:** Node.js (Express) server managing data flow and process execution.
- **Terminal:** Responsive HTML5/CSS3/JS dashboard.

## Setup
1. `pip install httpx`
2. `npm install`
3. `node server.js`
